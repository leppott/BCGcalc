The BCG style thermal model uses the same style files as the BCG calculation.

A single zip file will be available for download. It will include all result files along with the original data file. All result data files will be CSV format.

The zip file will included the base name of the original file along with the date and time.

The individual files will include the same basename along with a suffix to represent each step of the calculation process.

As an example if the input file was “xxx.csv” the result files will look like those in the table below.

Calculation, Thermal Model, zip file contents.
Output file Description
xxx original input file
xxx_modtherm_1markexcl original input file with an ‘EXCLUDE’ taxa column added in. It is a TRUE/FALSE field in which R marks redundant taxa as ‘TRUE’ based on the phylogeny fields in the input file. All ‘Exclude’ decisions are sample-specific. A detailed description of the process that the R tool uses for marking redundant taxa is available upon request (Jen.Stamp@tetratech.com)
xxx_modtherm_2metval_all values for all bug metrics in the BioMonTools R package (https://github.com/leppott/BioMonTools)
xxx_modtherm_2metval_BCG values for bug metrics used in the Thermal Model (model + flags); the calculations are done with the BioMonTools R package (https://github.com/leppott/BioMonTools)
xxx_modtherm_3metmemb values for bug metrics used in the MaritimeNW BCG model; each metric is assigned to assigned metric membership values (0-1)
xxx_modtherm_3metrules Rules for each class x BCG level (metrics, thresholds, symbols). Rules are similar but not exactly the same. long data format
xxx_modtherm_4levmemb for each sample, membership in each BCG level (0-1). Includes Index_Name and Index_Class
xxx_modtherm_5levassign results without the flags (primary, secondary levels and membership, continuous score, narrative with and without pluses and minuses)
xxx_modtherm_6levflags summary of how many samples had flagged metrics (and not flagged) based on the input file; count of # flagged (flag) and not flagged (NA)
xxx_modtherm_6metflags results of flag calculations for all metrics and samples
xxx_modtherm_RESULTS (csv) for each sample, thermal model output (primary, secondary levels and membership, continuous model score, narrative with and without pluses and minuses) plus flags (flag or NA)
xxx_modtherm_RESULTS (html) histogram showing distribution of samples across Thermal Model levels.
