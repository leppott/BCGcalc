The thermal model uses the same style files as the BCG calculation.

Required Fields

The following fields are required in input files.

Click here to download an Excel file with this table plus example data.

Table 1. Required fields.
Column Name Data Type Allowable entries Comment
Index_Name text Therm_ORWA_Bugs500ct Directs the R code to the correct overall model to use (it references the INDEX_NAME column in the Rules table in the BCGcalc extdata folder)
Index_Class text ORWA Directs the R code to the correct submodel to use (it looks in the INDEX_CLASS column in the Rules table in the BCGcalc extdata folder)
SAMPLEID text NA Unique sample identifier
TAXAID text NA Taxonomic identification of organisms in each sample
N_TAXA numeric non-negative real number Number of individuals.
EXCLUDE text TRUE, FALSE Redundant (non-distinct) taxa should marked “TRUE”. Redundant taxa are excluded from richness metric calculations but are included in the composition (percent individuals) metrics.
NONTARGET text TRUE, FALSE Non-target taxa should be marked “TRUE”. The R code excludes non-target taxa from all metric calculations.
PHYLUM text NA taxonomic hierarchy
CLASS text NA taxonomic hierarchy
SUBCLASS text NA taxonomic hierarchy
ORDER text NA taxonomic hierarchy
FAMILY text NA taxonomic hierarchy
SUBFAMILY text NA taxonomic hierarchy
TRIBE text NA taxonomic hierarchy
GENUS text NA taxonomic hierarchy
THERMAL_INDICATOR text stenoc, cold, cool, cowa, warm, stenow, eurythermal Thermal preference assignment based on Stamp et al. (in progress): stenoc=cold stenotherm; cold, cool, cowa = cool/warm, warm, stenow = warm stenotherm, eurythermal.

Optional Fields

Fields necessary to calculate other metrics can be included and do not affect the thermal model output. Additional metrics are available in the ’*metval_all.csv’ file.

Table 2. Optional fields
Column Name Data Type Allowable entries Comment
BCG_ATTR text 1i, 1m, 2, 3, 4, 5, 6 Biological Condition Gradient (BCG) attribute assignments
FFG text CG, CF, PR, SC, SH, MH, OM, PA, PH, XY Primary Functional Feeding Group. Multiple entries are allowed and should be separated with a comma (and no space; e.g., CG,CF). Entries: collector-gatherer (CG), collector-filterer (CF), predator (PR), scraper (SC), shredder (SH), macrophyte herbivore (MH), omnivore (OM), parasite (PA), piercer-herbivore (PH), xylophage (XY).
LIFE_CYCLE text uni, semi, multi Number of broods or generations of an organism in a year (also referred to as ‘voltinism’): univoltine (UNI), semivoltine (SEMI), or multivoltine (MULTI).
TOLVAL numeric number ranging from 0-10 Taxa with tolerance values ranging from 0 to 3 are considered ‘intolerant’ to anthropogenic disturbance and are included in the ‘intolerant’ metric calculations. Those with values ranging from 7 to 10 are included in the ‘tolerant’ metrics.
HABIT text BU, CB, CN, SP, SW Habit. Multiple entries are allowed and should be separated with a comma. Entries: burrowers (BU), climbers (CB), clingers (CN), sprawlers (SP), swimmers (SW).
NOTEWORTHY text TRUE, FALSE Noteworthy taxa occur most often at sites with complex and resilient habitat structure that supports high overall freshwater biodiversity. Most of these taxa are rarely or occasionally encountered in benthic macroinvertebrate biomonitoring samples.
LONGLIVED text TRUE, FALSE These invertebrates require more than one year to complete their life cycles; thus, they are exposed to all the human activities that influence the stream throughout one or more years. If the stream is dry part of the year or subject to flooding, these animals may disappear. Loss of long-lived taxa may also indicate an on-going problem that repeatedly interrupts their life cycles
